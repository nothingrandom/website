Katy Hollway

Katy Hollway is a teen fiction author. This project involved designing a website that would captivate her audience, and later design a book cover.

Date

January 2014

Role

Design & Development

URL

www.katyhollway.com

The Design

The project design started off as a relatively simple website. I decided that only a one page website was required, and worked out the best way to display the information. I split the information into three primary sections, and designed them all slightly differently while maintaining an overall design.

Color scheme and fonts

The color pallet I used is composed of a range of blue and grey hues. This helps to draw attention towards the website content. Both the fonts that I used are available for free, and are quite different from each other. This is so that they are easily distinguishable on both the website and on print.

The development

After development on the website started, I realized that I needed to add something to make the website stand out more. So I wrote a script to give sections of the website a parallax effect. I also wrote a script to make the header "sticky".

Making it mobile

The mobile site was easy to implement. As the images are rather minimal and there is only one page, not a lot of optimization was required in order to get the site to load quickly. The website has been viewed on multiple devices with a range of different internet speeds, so spending a little bit of extra time on the mobile website was worthwhile.
